Use induction to prove that \(4\) divides \((5^n -1)\) for \(n \geq 1\text{.}\)

  1. Base Case: (Check for \(n=1,2,3\) to make sure you believe it. You actually only need to check for \(n=1\text{.}\))

  2. Inductive Hypothesis: Write down your inductive hypothesis for \(k\text{.}\)

    Pro Tip: A nice way to write "4 divides \((5^k-1)\)" is the statement "\(5^k-1 = 4r\) for some \(r\text{.}\)"

  3. Inductive Step: Write down what you want to SHOW for \(k+1\text{.}\) Then try to find "a copy of the size \(k\) problem" in the size \(k+1\) problem. Rearrange and use your inductive hypothesis to prove the result!

    Hint: For this problem, the easiest thing to do is "add 0 in a smart way." Namely, add \((5^k - 5^k)\) and then rearrange.
