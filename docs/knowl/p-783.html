Here is Hall's Matching Theorem: Given a bipartite graph with \(V = A \cup B\) and \(|A|=|B|\text{.}\) Then \(G\) has a perfect matching \(\Longleftrightarrow\) for every \(S \subset A\text{,}\) we have \(|S| \leq |N(S)|\text{.}\) The last statement is called Hall's Condition. It means that "\(G\) has no bottlenecks."

  1. Let \(G=(V,E)\) be a \(d\)-regular bipartite graph. By Problem 2, we have \(|A|=|B|\text{.}\) You will show that Hall's condition holds for \(G\text{.}\) This means that a \(d\)-regular bipartite graph always has a perfect matching.

    1. Pick any subset \(S \subset A\text{.}\) What is \(\sum_{v \in S} \deg(v)\text{?}\)

      \begin{equation*} \sum_{v \in S} \deg(v) = d |S| \end{equation*}

    2. Explain why \(\displaystyle{\sum_{v \in S} \deg(v) \leq \sum_{v \in N(S)} \deg(v)}\text{.}\)

      Every edge from a vertex in set \(S\) ends at a vertex in \(N(S)\text{.}\) This confirms that

      \begin{equation*} \sum_{v \in S} \deg(v) \leq \sum_{v \in N(S)} \deg(v). \end{equation*}

      Note that this inequality can be strict: vertices in \(N(S)\) may have neighbors outside of the set \(S\text{.}\)

    3. Why does part (b) guarantee that \(|S| \leq |N(S)|\text{?}\)

      \begin{equation*} d |S| = \sum_{v \in S} \deg(v) \leq \sum_{v \in N(S)} \deg(v) = d|N(S)| \end{equation*}

      and so \(|S| \leq |N(S)|\text{.}\)

  2. Here is a 3-regular bipartite graph \(G\) on 6 vertices.

    1. Find a perfect matching \(M\) in the graph \(G\text{.}\)

    2. Now draw the graph \(H=G-M\) where you remove the edges in your matching \(M\text{.}\)

    3. Find a perfect matching in \(H\text{.}\) Why does Problem 5 guarantee that \(H = G-M\) has a perfect matching?

      \(G-M\) is a 2-regular graph. So it has a perfect matching.

  3. Let \(G\) be a \(d\)-regular bipartite graph where \(|A|=|B|=n\text{.}\) Use (regular) induction on \(d \geq 1\) to prove that a \(d\)-regular bipartite graph has \(d\) disjoint perfect matchings. This means that we can color the edges of \(G\) using \(d\) colors so that every vertex is incident with exactly one edge from each of the \(d\) colors.

    • Base Case: \(d=1\text{.}\)

      A 1-regular graph is a perfect matching!

    • Inductive Hypothesis: Assume that if \(G\) is a \((d-1)\)-regular bipartite graph where \(|A|=|B|=n\text{,}\) then \(G\) has \(d-1\) disjoint perfect matchings. \((\star)\)

    • Inductive Step: Use \((\star)\) to prove that if \(G\) is a \(d\)-regular bipartite graph where \(|A|=|B|=n\text{,}\) then \(G\) has \(d\) disjoint perfect matchings.

    By problem 5, The \(d\)-regular bipartite graph \(G\) has a perfect matching \(M\text{.}\) Let's remove these edges. The graph \(G-M\) is \((d-1)\)-regular. By \((\star)\text{,}\) \(G-M\) has \(d-1\) disjoint perfect matchings. So we have found \(d\) disjoint perfect matchings.
