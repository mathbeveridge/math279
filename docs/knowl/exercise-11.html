Exercise 1. Element, Subset or Neither?

Mark each of the following statements as true or false

  1. Let \(A = \{ 1, \{ 1 \} \}\text{.}\)

    1. \(1 \in A\) Solution.

    2. \(\{ 1 \} \in A\) Solution.

    3. \(\{ \{ 1 \} \} \in A\) Solution.

  2. Let \(B = \{ 1, 2, 3 \}\)

    1. \(3 \in B\) Solution.

    2. \(\{ 3 \} \in B\) Solution.

    3. \(\{ 3 \} \subseteq B\) Solution.

  3. \(C = \{ \{1,2\}, \{ \emptyset \} \}\)

    1. \(\emptyset \in C \) Solution.

    2. \(\emptyset \subseteq C \) Solution.

    3. \(|C|=3\) Solution.

  4. \(D = \{ \{1, 2 \}, \{ 3, 4, 5 \} \}\)

    1. \(|D|=5 \) Solution.

    2. \(\{ 1, 2 \} \subseteq D \) Solution.

    3. \(4 \in D\) Solution.

  5. \(E = \{ \emptyset, \{ \emptyset \} , 0 \}\)

    1. \(\emptyset \in E\) Solution.

    2. \(\{ \{ \emptyset \} \} \subseteq E \) Solution.

    3. \(|E|=3\) Solution.
