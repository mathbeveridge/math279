Solution 1.
After reserving 16 fries for Anna, we have 34 fries left over, to distribute any way we like. The number of ways is \({34+4-1 \choose 50} = {37 \choose 34}\text{.}\) The same argument holds for Boris, Clyde and Denise. so
\begin{equation*} |A|+|B|+|C|+|D| = 4 {37 \choose 34}. \end{equation*}
