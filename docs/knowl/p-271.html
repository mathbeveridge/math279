The identity function \(\mbox{Id}_X: X \rightarrow X\) is given by \(\mbox{Id}_X(x) = x\text{.}\) (So every element just maps to itself.) Let \(f : X \rightarrow Y\) and \(g: Y \rightarrow X\) be functions so that \(g \circ f = \mbox{Id}_X\text{.}\)

  1. Give an example where \(g \circ f = \mbox{Id}_X\) but \(f \circ g \neq \mbox{Id}_Y\text{.}\) Solution.

  2. Prove that if \(g \circ f = \mbox{Id}_X\) then \(f\) is injective. Solution.

  3. Prove that if \(g \circ f = \mbox{Id}_X\) then \(g\) is surjective. Solution.
