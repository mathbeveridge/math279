Solution 1.
After reserving 16 fries for Anna, and 16 fries for Boris, we have 18 fries left over, to distribute any way we like. The number of ways is \({18+4-1 \choose 18} = {21 \choose 18}\text{.}\) The same argument holds for any pair of friends, so
\begin{equation*} | A \cap B| + | A \cap C| + | A \cap D| + | B \cap C| + | B \cap D| + | C \cap D|= 6 {21 \choose 18}. \end{equation*}
