  • For \(n=3\) we have \(\sum_{i=0}^{3} 2^i = 2^0 + 2^1 +2^2 + 2^3= 1 + 2 + 4 +8 = 15\) and \(2^4 -1 = 16 -1 =15\text{.}\) So the statement is true for \(n=3\text{.}\)
