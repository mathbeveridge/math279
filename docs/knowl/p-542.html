\begin{equation*} v_G -e_G +f_G = 2. \end{equation*}

  1. Base Case: \(k=0\). A connected graph with no cycles is a tree. We know that a tree on \(v_G\) vertices has \(v_G-1\) edges. Explain why \(v_G -e_G +f_G = 2.\)
