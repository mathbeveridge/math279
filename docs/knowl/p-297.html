Let \(f : X \rightarrow Y\) be a function. Let \(A \subseteq X\) and \(B \subseteq X\text{.}\) For a subset \(S \subseteq X\text{,}\) define the subset \(f(S) \subseteq Y\) by

\begin{equation*} f(S) = \{ y \in Y \mid \exists s \in S, f(s) = y \} \end{equation*}

  1. Prove that if \(A \subseteq B\) then \(f(A) \subseteq f(B)\) Solution.

  2. Draw an example for which \(A \not\subseteq B\text{,}\) but \(f(A) \subseteq f(B)\text{.}\) Solution.

  3. Prove that \(f(A \cap B) \subseteq f(A) \cap f(B)\text{.}\) Solution.

  4. Draw an example for which \(f(A \cap B) \neq f(A) \cap f(B)\) Solution.
