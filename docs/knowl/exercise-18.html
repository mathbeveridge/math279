Exercise 3. Distributive Laws.

Prove the following distributive laws in two ways. First, using a Venn diagram, and then using a double inclusion proof.

  1. \(\displaystyle A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\)

  2. \(\displaystyle A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\)
